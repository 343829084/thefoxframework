# TheFox Cpp Framework

## 简介 ##
开源的C++的基础开发库，包含包括常见的C++常用工具类（file、mutex、semaphore、thread、线程池等）、mysql封装、net-snmp封装、net库、rpc、log库。

现在主要工作平台是Windows，最近也在将其中的一部分慢慢的实现Windows和Linux跨平台

本库尽量保证轻量级，功能比较简单

依赖库： 基本上都需要的支持std::tr1, rpc部分需要protobuf库
